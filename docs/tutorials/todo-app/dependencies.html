Learn to create a simple Todo App with Metal.js

Dependencies

In order to follow along with this tutorial, you'll need Git, and a recent version of Node.js/npm.

Todo Boilerplate

Start by cloning the boilerplate respository.

git clone https://github.com/Robert-Frampton/metal-todo-app.git

Then navigate to the root of the project in your terminal and install the local npm dependencies.

cd metal-todo-app && npm install

The boilerplate that you will be building off of is located in the src/jsx directory. You can also view the finished product in the src/jsx/final directory.

Building

In order to test drive the boilerplate located in the src/jsx directory, you'll need to run the provided build script:

npm run build:jsx

This will run babel + webpack to transpile and bundle the JavaScript into something consumable by web browsers.

Everything you'll be writing in this tutorial will be using ES6 syntax, if you're unfamiliar with it, take a look at this guide first.

To build the finished product, run the following script:

npm run build:jsx:final

Demo

Now that you've built the project, go ahead and open the demo page located at demos/index.html.

If you take a look at the file you'll see that the component is already being invoked.

<script type="text/javascript">
    new metal.TodoApp();
</script>

When invoking a component this way, the component is rendered to the body element.
